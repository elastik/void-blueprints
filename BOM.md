## 4. Cost Analysis Summary

### 4.1 Tooling Costs (One-Time, Amortized)

These are upfront fixed costs required before production begins. They are NOT included in per-unit COGS above but must be recovered through unit sales.

| Tooling Item | Estimated Cost | Notes |
|---|---|---|
| Dome shell injection mold | $8,000 - $12,000 | Single-cavity P20 steel mold. Polycarbonate requires hardened steel. Shared across Void + Dark Dome. |
| Base housing injection mold (Void / Dark Dome) | $6,000 - $10,000 | Single-cavity, ABS. Less complex than dome (no optical clarity needed). |
| Base housing injection mold (Void Core) | $8,000 - $12,000 | Separate mold for taller base with reservoir chamber. Only needed at 350+ Kickstarter backers. |
| PCB design & prototyping | $500 - $1,000 | One-time engineering cost. |
| Packaging design & plates | $500 - $1,000 | One-time creative / print setup. |
| **Total tooling (Void + Dark Dome)** | **$15,000 - $24,000** | Minimum to start Kickstarter fulfillment |
| **Total tooling (all variants)** | **$23,000 - $36,000** | Including Void Core base mold |

**Tooling amortization at 1000 units:**

Using midpoint estimates ($19,500 for Void + Dark Dome tooling):

| Variant | Tooling Share | Amortized Per Unit (1000 total units) |
|---|---|---|

| Dark Dome | ~20% of dome + base mold (shares molds) | $3.90 |
| Void Core | ~20% of shared + own base mold | $7.80 + $10.00 = $17.80 |

### 4.4 Kickstarter Economics

Kickstarter takes ~5% platform fee + ~3-5% payment processing. Using 8% combined.

| Metric | The Void ($59 EB) | Void Core ($249 KS tier) | Dark Dome (stretch goal) |
|---|---|---|---|
| **Kickstarter price** | $59.00 | $249.00 | $39.00 (est.) |
| **After KS fees (8%)** | $54.28 | $229.08 | $35.88 |
| **COGS (1000pc + tooling)** | $28.95 | $80.05 | $12.15 |
| **Net margin** | **$25.33 (43%)** | **$149.03 (60%)** | **$23.73 (48%)** |
| **Shipping estimate** | $8.00 (domestic) | $10.00 (domestic) | $7.00 (domestic) |
| **Net after shipping** | **$17.33 (29%)** | **$139.03 (56%)** | **$16.73 (34%)** |

**Kickstarter viability assessment:**

- **The Void at $59 early bird:** Viable. After all fees and shipping, ~29% net margin. Not huge, but positive. Volume improvements at 2000+ units would push COGS down further. The $59 price point works as a customer acquisition tool -- recurring Void Pack revenue is the real margin driver.
- **Void Core at $249:** Very comfortable. 56% net margin even after fees and shipping. The Pi Zero at $15 is the cost floor but still leaves excellent margin.
- **Dark Dome at ~$39 (stretch goal estimate):** Viable. 34% net margin after fees and shipping. The near-zero electronics cost is the key advantage. Could potentially go as low as $35 and still be marginally positive.

### 4.5 Breakeven Analysis

| Variant | Fixed Tooling Cost | Net Margin Per Unit (KS price after fees + shipping) | Units to Break Even on Tooling |
|---|---|---|---|
| The Void | $17,000 (share of molds) | $17.33 | ~981 units |
| Void Core | $20,000 (share + own mold) | $139.03 | ~144 units |
| Dark Dome | $5,500 (share of molds) | $16.73 | ~329 units |

**The Void needs ~1000 backers to break even on tooling alone. This should be the Kickstarter funding goal calculation basis.** Void Core breaks even very quickly due to high per-unit margin. Dark Dome is a stretch goal and needs ~330 units.

## 5. Sourcing Notes

### 5.1 Preferred Suppliers

| Category | Primary Source | Secondary Source | Notes |
|---|---|---|---|
| **Passive ICs (555, drivers, PD sink)** | LCSC Electronics | Digi-Key (reference/validation) | LCSC has best pricing for Chinese-manufactured equivalents. Order with PCBs from JLCPCB for combined shipping. |
| **LEDs (UV-A, Blue, RGB)** | AliExpress (100pc+ packs) | LCSC for SMD reels | 395nm and 450nm 5050 LEDs are commodity parts. Multiple redundant sources. |
| **UV-C LED (275nm)** | AliExpress (Seoul Viosys compatible) | LED Supply (ledsupply.com) for branded Seoul Viosys | UV-C LEDs are specialty but increasingly available. Verify radiant power with datasheets. |
| **PCBs + Assembly** | JLCPCB (PCB + SMT assembly) | PCBWay | JLCPCB offers combined PCB fabrication + component sourcing + SMT assembly. Best value for <5000 units. |
| **Injection molding** | China contract manufacturer (Alibaba RFQ) | US domestic (Protolabs, Xometry) for prototyping | China offers 50-70% cost savings on tooling and per-unit. US domestic for speed on prototypes. |
| **Raspberry Pi Zero W 2** | Raspberry Pi Foundation authorized distributors | Pimoroni, Adafruit, SparkFun | Fixed $15 pricing. Verify stock availability before committing. Lead time can be 8-16 weeks when constrained. |
| **Sensors (BME280)** | AliExpress modules | Digi-Key (Bosch genuine) | AliExpress modules are $2-3 ea. Genuine Bosch chips are $2-4 on Digi-Key. Module includes breakout board. |
| **Packaging** | PakFactory / Packlane | Local print shop for prototypes | UV-reactive ink adds ~$0.25-0.50/unit over standard printing. Get samples before committing. |

### 5.3 Minimum Order Quantities

| Component | MOQ | Cost Impact |
|---|---|---|
| Injection molding (per run) | 500-1000 units typical | Below 500, per-unit cost increases significantly |
| PCBs (JLCPCB) | 5 units | No practical constraint |
| PCB assembly (JLCPCB SMT) | 2 units | No practical constraint |
| LEDs (AliExpress) | 10-100 units per pack | No practical constraint |

| Custom packaging | 100-500 units | Most packaging suppliers have 100-250 unit MOQ |

### 5.4 Cost Reduction Opportunities

| Opportunity | Potential Savings | Feasibility | Notes |
|---|---|---|---|
| Replace ATtiny85 + 555 timers with single ATtiny85 | $0.10-0.25/unit | High | ATtiny85 can replace both 555 timers in firmware. Reduces component count. |
| ESP32-S3 instead of Pi Zero W 2 (Void Core) | $10-12/unit | Medium | ESP32-S3 is $3-5 vs $15 for Pi Zero. Trades Linux ecosystem for cost savings. Requires firmware rewrite. NEEDS VALIDATION. |
| MH-Z19B instead of SCD40 (Void Core CO2) | $5-8/unit | Medium | MH-Z19B is $5-8 vs $12-18 for SCD40. Larger physical package but adequate accuracy. |
| Multi-cavity injection mold | $1-2/unit at 5000+ | Medium | Doubles tooling cost but halves cycle time and per-unit cost. Worth it at 5000+ units. |
| Eliminate UV-reactive packaging ink | $0.25-0.50/unit | Low priority | UV-reactive ink is a brand-defining feature. Only cut if margins are critically tight. |
| Shared mold for Dark Dome (no separate base mold) | $6,000-10,000 tooling savings | High | Use same base mold as The Void. Dark Dome base just has empty electronics bay. Minor material waste but huge tooling savings. |
